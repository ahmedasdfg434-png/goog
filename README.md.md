# 🔐 نظام تغيير كلمة المرور - نسخة منفصلة

نظام تجريبي مستقل تماماً لمحاكاة واجهة تغيير كلمة المرور مع لوحة تحكم للأدمن.

## ✨ المميزات

- ✅ **مستقل تماماً** - كل ملف يعمل لوحده
- ✅ **تخزين محلي** - جميع البيانات في متصفحك فقط
- ✅ **لا يحتاج خادم** - يعمل على GitHub Pages أو أي استضافة مجانية
- ✅ **واجهة عربية** - دعم كامل للغة العربية
- ✅ **آمن للتعلم** - كلمة مرور للوحة التحكم

## 📁 الملفات

| الملف | الوصف |
|-------|-------|
| `index.html` | الصفحة الرئيسية - روابط سريعة |
| `google-password-change2.html` | صفحة تغيير كلمة المرور (للمستخدم) |
| `admin.html` | لوحة تحكم الأدمن (محمية) |

## 🔧 طريقة العمل

### للمستخدم (صفحة تغيير كلمة المرور)
1. افتح `google-password-change2.html`
2. أدخل البريد الإلكتروني وكلمات المرور
3. اضغط "تغيير كلمة المرور"
4. البيانات تُحفظ تلقائياً في متصفحك

### للأدمن (لوحة التحكم)
1. افتح `admin.html`
2. أدخل كلمة المرور: `admin@123`
3. استورد البيانات من:
   - `user_data` - البيانات الأساسية
   - `backup_data` - نسخة احتياطية
4. شاهد جميع البيانات المدخلة
